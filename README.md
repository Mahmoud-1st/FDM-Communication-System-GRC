# FDM Communication System Simulation using GNU Radio

## Overview

This project demonstrates Frequency Division Multiplexing (FDM) using GNU Radio Companion (GRC).

The design was developed as part of my Communication Theory studies to better visualize signal processing in the Physical Layer.

## Concepts Covered

- Frequency Division Multiplexing (FDM)
- AM Modulation
- AM Demodulation
- Superheterodyne Receiver
- Frequency Translation
- Filtering
- Spectrum Analysis

## Tools

- GNU Radio Companion (GRC)

## Learning Outcomes

Through this project I gained practical experience with:

- Signal flow visualization
- Communication system architecture
- Mixer operation
- Intermediate Frequency (IF)
- Receiver design concepts
